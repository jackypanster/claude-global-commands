## 冲突状态
- Git 状态：!`git status`
- 冲突文件：!`git diff --name-only --diff-filter=U`

冲突解决助手：

1. **分析冲突**
   - 识别冲突类型和原因
   - 分析冲突双方的意图
   - 评估冲突复杂度

2. **解决策略**
   - 提供解决建议
   - 保留关键功能
   - 确保代码一致性

3. **验证解决**
   - 检查语法正确性
   - 建议测试验证
   - 确认提交准备

冲突背景：$ARGUMENTS
